Permuted Index

The permuted index is generated from the ordinary index files (those including Common Graphics). Since none of the circle drawing functions in Common Graphics have names starting with "circle", they are hard to find in the regular index but all appear under "circle" in the Permuted Index. Like the index, the permuted index is divided into single-letter pages, with an additional page for non-alphabetic characters.

In order to suppress line wrapping (which makes entries difficult to read), certain entries have been shortened by replacing text with abbreviations (IDE for Integrated Development Environment, e.g.) or replacing elements of a list of functions (typically in ANSICL headings) with [...].
