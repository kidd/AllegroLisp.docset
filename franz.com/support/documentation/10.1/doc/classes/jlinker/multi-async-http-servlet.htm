multi-async-http-servlet

This class and the related async-http-servlet class are classes that should be subclassed by an application. The subclass defines working methods for the generic functions listed in The jLinker Servlet API. The subclass also defines a value for the java-classes slot.

This class is defined when the jlinkent module is loaded. See The jLinker Servlet API for details on the jLinker Servlet API.

See jlinker.htm for more information.
