device-clear-output

Arguments: stream

If the device connected to stream has functionality to clear any pending output, it is called. Otherwise, no action is taken.

Defined methods

  • On class: simple-stream: returns nil.

See streams.htm for information on the simple-streams implementation in Allegro CL. The older but still supported Gray streams implementation is described in gray-streams.htm.
