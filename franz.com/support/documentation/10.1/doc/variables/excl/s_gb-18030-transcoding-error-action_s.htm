*gb-18030-transcoding-error-action*

The value of this variable determines how the :gb18030 external-format deals with transcoding errors. It is one of the following values:

  • nil: Ignore transcoding errors silently.
  • :warn: Signal a warning of class gb-18030-transcoding-warning.
  • :count: Replace the value of this variable with a count of transcoding errors.
  • an integer: Same as :count but continue counting from the given integer value.
  • :collect: Replace the value of this variable with a list of error markers. For each error signalled while converting characters to octets, push the offending character onto the list.
    For each input error, push a list of the offending octets onto the list.
  • a cons: Same as :collect but continue pushing onto the given list.
  • :error (or any other value not above): Signal an error of class gb-18030-transcoding-error.

In all cases insert the gb-18030-transcoding-error-char character to replace the offending character or octets in the transcoded stream.

See iacl.htm for more information on international characters and external formats.
