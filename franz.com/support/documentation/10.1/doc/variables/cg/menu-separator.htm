menu-separator

The value is a menu-item which when added to a pop-up or pull-down menu gives an unselectable dashed line.

GTK Note: this special value should not be used on GTK in a menu where the set of items may be changed by calling (setf menu-items) or update-menu. Instead you should use a menu-item whose title is a string of a single dash character (i.e. "-"), which accomplishes the same thing. For example: (make-instance 'menu-item :title "-"). This makes every menu-item be a distinct object, as needed on GTK for replacing menu-items.
