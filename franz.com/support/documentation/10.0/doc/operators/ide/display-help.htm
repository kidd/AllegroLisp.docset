display-help

Arguments: &optional symbol-or-string

Takes a symbol or a string as its only argument, and attempts to display online documentation for it. The Help | Help On Selected Symbol command calls this function with the currently selected symbol.

If the argument is a string, it should be the title of one of the essay files describing Allegro CL or Common Graphics. Currently this is used by the Help | Help On Selected Dialog command to display the topic associated with each IDE dialog.
