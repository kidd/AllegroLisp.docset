short-string

Arguments: x

Returns a short string constructed from x, analogous to string. x can be a short string, a regular string, a symbol of a character. Here is the action of short-string in each case:

  • x is a short string: x is returned.
  • x is a regular string: a new short string is created containing all the characters in x (up to the fill pointer, if there is one) and the new short string is returned.
  • x is a character: a new short string of length 1 containing that character is returned.
  • x is a symbol: a new short string is constructed from the print-name of x and returned.

See Arrays and short arrays in implementation.htm for a definition of short arrays and a discussion of the array implementation in Allegro CL.
