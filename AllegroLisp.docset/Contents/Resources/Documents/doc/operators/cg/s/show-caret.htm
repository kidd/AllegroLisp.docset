show-caret

Arguments: window

Enables the caret of window if it has been disabled with hide-caret and currently has the keyboard focus.

GTK Note

Carets have not yet been implemented on the GTK platform. Calling caret-related functions on GTK will have no effect, except that show-caret and hide-caret will invoke the *gtk-compatibility-warning-action*.
