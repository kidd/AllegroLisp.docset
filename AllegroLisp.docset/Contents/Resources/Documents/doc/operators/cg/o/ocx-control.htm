ocx-control

Arguments: ocx-widget-window

Applies to the Windows platform only.

Returns an integer that is the handle to the OCX control that has been placed into an ocx-widget-window, where the ocx-widget-window is the "site window" for the OCX control. (The function cg:handle returns the handle of the site window.) These handles may be useful if you need to use lower-level OLE functionality on the control.
