lisp-value

Arguments: index

This returns the value in the table of registered values (see register-lisp-value) at the given index.

This function is setf'able. Thus, one can replace slots in the foreign values table.

This function is the Lisp equivalent of the C function lisp_value provided for referencing Lisp values from foreign code. See Accessing Lisp values from C: lisp_value() in foreign-functions.htm.

See ftype.htm for information on foreign types in Allegro CL and foreign-functions.htm for general information on foreign functions in Allegro CL.
