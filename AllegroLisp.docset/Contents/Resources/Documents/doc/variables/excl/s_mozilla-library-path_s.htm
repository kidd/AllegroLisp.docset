*mozilla-library-path*

Applies to GTK IDE/CG implementations only.

On GTK, the Common Graphics html-widget uses a GTK control supplied by Mozilla. This variable needs to contain the directory where this control is installed for the html-widget to work (including the Help Window in the IDE).

We have no way to determine the installation directory automatically. If you attempt to show an html-widget and the value of this variable is not set properly (or the Mozilla control is not installed), then an informational error will be signaled. You can attempt to locate the directory by calling find-mozilla-gtk-path.
