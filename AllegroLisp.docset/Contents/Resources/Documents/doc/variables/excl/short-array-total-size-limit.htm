short-array-total-size-limit

The maximum total size of a short array. The value is (expt 2 24) in a 32-bit lisp, and (expt 2 56) in a 64-bit lisp. (In releases prior to 7.0, this value was the maximum number of dimensions of arrays in general.) See Arrays and short arrays in implementation.htm for a definition of short arrays and a discussion of the array implementation in Allegro CL.
